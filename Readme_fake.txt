# Fake News Classifier.

Fake News detection using PassiveAggresive Classifier.

### Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

To get things started you need to install following softwares and packages:
```
1.Jupyter notebooks
2.Python3
```

### Installing
Anaconda:
```
Go to the Anaconda Website and choose a Python 3.x graphical installer according to the Operating system you are using.
```
```
Remaining Packages can be installed with pip package provided by the Anaconda software 
```

### Dataset
```
The data was provided by Kaggle and I added a prediction block in the code where you can upload your own dataset or you can write your own text.
```


### Models used
```
Machine learning models:
PassiveAggressive Classifier

```
